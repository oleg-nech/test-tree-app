## test-tree-app
Test tree application

### Requirements 
```
node.js
grunt-cli
bower
```

## Instalation
```
npm install
bower install
```

## Run server
```
grunt start
```
Open [http://localhost:5005](http://localhost:5005)

## Tests
### From console
```
grunt test
```

### From browser
```
grunt start
```
Open [http://localhost:5005/test](http://localhost:5005/test)
