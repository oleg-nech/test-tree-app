## test-tree-app
Test tree application

### Requirements 
```
node.js
grunt-cli
bower
```

### Used frameworks and technologies
```
node.js
express.js
grunt
npm
bower
backbone.js
marionatte.js
ES6 - Babel.js
browserify
bootstrap
less
mocha.js
chai.js
sinon.js
```

## Instalation
```
npm install
bower install
```

## Run server
```
grunt start
```
Open [http://localhost:5005](http://localhost:5005)

## Tests
### From console
```
grunt test
```

### From browser
```
grunt start
```
Open [http://localhost:5005/test](http://localhost:5005/test)
